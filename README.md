# Hi there, I'm Sarker Ayon
### Full-Stack Developer & Digital Architect

I bridge the gap between complex backend engineering and minimalist, premium UI/UX. As a developer based in Bangladesh—and a student at Rajuk Uttara Model College—I focus on building scalable SaaS ecosystems and high-performance digital products that are as intuitive as they are technically robust. 

Whether I'm architecting a luxury storefront or a complex restaurant management ecosystem, I build with performance, precision, and a deep appreciation for clean layouts.

---

### 🔭 What I'm Currently Working On
*   **Architecting Platforms:** Developing full-stack solutions including **Scentscape** (e-commerce), **Menuflow** (executive dashboards), **Tickstick** (event ticketing), and **POS Guru**.
*   **Workflow Automation:** Building out efficient server-side scripts and CSV processing pipelines using **Python** and **n8n**.
*   **Community Building:** Organizing and executing technical infrastructure for events like **Rajuk National SciSpark 4.0**.
*   **Deep Dives:** Mastering advanced React patterns, PostgreSQL optimization, and motion design for seamless user journeys.
